Guide d'utilisation

Tout ce qu'il faut savoir pour bien utiliser MIFFA

Démarrage Ma famille Relations Modules Invités Confidentialité FAQ

Démarrage

1

Créez votre compte

Choisissez si vous voulez créer une nouvelle famille (en tant qu'Administrateur) ou rejoindre une famille existante (en tant que Membre).

2

Administrateur : créez votre famille

Donnez un nom à votre famille. Un code unique est généré automatiquement — partagez-le avec vos proches pour qu'ils puissent vous rejoindre.

3

Membre : rejoignez une famille

Entrez le code fourni par votre administrateur. Votre demande sera envoyée et devra être approuvée avant que vous n'accédiez aux fonctionnalités.

4

En attendant l'approbation

Vous pouvez modifier votre profil et vous déconnecter. Vous pouvez aussi envoyer un rappel à l'administrateur si la demande prend du temps.

Ma famille

Code de la famille

Visible dans le panneau d'administration. Cliquez sur le code pour le copier et le partager facilement.

Bannière

L'administrateur peut personnaliser la bannière affichée en haut du tableau de bord de tous les membres.

Ajouter un membre

L'admin peut envoyer une invitation directe par nom d'utilisateur — la personne doit l'accepter pour rejoindre.

Retirer un membre

Le retrait d'un membre passe par un vote de tous les membres de la famille, pas une décision unilatérale.

Définir vos relations

Une fois approuvé(e), rendez-vous sur « Mes relations » pour indiquer votre lien avec les membres de votre famille (parent, enfant, conjoint, frère/sœur...).

  • La personne concernée reçoit une notification et un email pour confirmer la relation.
  • Une relation ne peut pas être définie deux fois avec la même personne.
  • Pour modifier ou supprimer une relation existante, l'accord des deux personnes est nécessaire.
  • Vos relations alimentent automatiquement l'arbre généalogique.

Les modules

Membres

Consultez les profils de tous les membres de votre famille.

Arbre généalogique

Visualisez les liens familiaux générés à partir des relations définies.

Chronologie

Naissances, mariages, diplômes... les grands événements de la famille classés par date.

Fil d'actualité

Partagez des nouvelles, photos et commentaires avec toute la famille.

Médiathèque

Créez des albums et stockez photos, vidéos et documents familiaux.

Calendrier

Organisez des événements et suivez les réponses (RSVP) des membres.

Statut Invité

Un invité a un accès temporaire et limité à une famille, défini par l'administrateur :

  • Une durée précise (ex : 7 jours)
  • Des modules spécifiques autorisés (membres, arbre, médiathèque...)
  • À l'expiration, l'accès est révoqué mais le compte reste actif

Confidentialité de votre profil

Dans « Modifier le profil », vous pouvez choisir de masquer certaines informations (profession, téléphone, ville, email...) aux autres membres de la famille grâce aux options « Rendre privé ».

Questions fréquentes

Ma demande d'adhésion n'est pas approuvée, que faire ?

Vous pouvez envoyer un rappel à l'administrateur depuis votre tableau de bord. Si la demande n'est toujours pas traitée après 24h, vous pouvez en renvoyer une nouvelle (jusqu'à 3 tentatives).

Puis-je appartenir à plusieurs familles ?

Vous avez une famille principale (en tant que membre ou admin), mais vous pouvez aussi être invité temporairement dans d'autres familles.

Comment supprimer mon compte ?

Depuis « Modifier le profil », cliquez sur « Supprimer mon compte ». Si vous êtes l'unique administrateur d'une famille avec d'autres membres, vous devez d'abord désigner un autre administrateur.

Comment fonctionne le vote de retrait d'un membre ?

Un administrateur initie le vote. Tous les autres membres (sauf la personne concernée) votent Oui ou Non. Si la majorité vote Oui, le membre est retiré de la famille — il garde son compte mais peut rejoindre une autre famille avec un code.
